# **App Name**: GuiaDoar

## Core Features:

- Donation Item Input: Allow users to input the type and description of items they wish to donate.
- Location Detection: Detect the user's current location to find nearby donation points.
- Nearest Points Identification: Identifies the nearest donation collection points, sorted by distance from the user.
- Point Information Display: Displays information about each donation point (address, opening hours, accepted items).

## Style Guidelines:

- Primary color: Teal (#008080) for trust and reliability.
- Background color: Light teal (#E0F8F8) for a clean and calm feel.
- Accent color: Soft blue (#77B5FE) for interactive elements.
- Headline font: 'Poppins' sans-serif for a modern, geometric style.
- Body font: 'PT Sans' sans-serif for readability.
- Use clear, universal icons to represent different donation categories.
- Simple and intuitive layout with clear visual hierarchy.
